## Definition
An MCP host is an LLM-powered application that requires or utilizes context from external systems, or needs to interact with them, often by using them as tools.

## Why it matters (in Poovi's context)
It represents the LLM-side of the MCP interaction, acting as the entity that consumes information and instructions from MCP servers.

## Key properties or components
- LLM-powered application
- Requires external context or interaction
- Can utilize external tools
