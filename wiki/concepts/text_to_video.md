## Definition
A type of AI video generation where a text description (prompt) is used as input to create a video sequence.

## Why it matters (in Poovi's context)
A primary capability discussed for tools like Runway Gen 3, enabling content creation from simple textual commands.

## Key properties or components

- Natural Language Processing
- Visual Synthesis
