## Definition
Container security refers to the practices and tools used to secure containerized applications and environments against threats. This includes vulnerability scanning, secure image building, and runtime protection.

## Why it matters (in Poovi's context)
A critical consideration when deploying containers, especially for sensitive applications or when using third-party images. Tools like Docker Scout help address this.

## Key properties or components
- Vulnerability scanning
- Image integrity
- Runtime monitoring
- Least privilege
