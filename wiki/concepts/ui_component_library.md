## Definition
A collection of reusable user interface elements and components designed to be used in software development. Libraries provide pre-built elements like buttons, forms, and navigation bars.

## Why it matters (in Poovi's context)
Used to significantly speed up the development process by providing ready-to-use, customizable UI elements.

## Key properties or components
- Reusability
- Consistency
- Customisation
- Pre-built components
