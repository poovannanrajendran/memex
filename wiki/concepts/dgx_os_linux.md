## Definition
A Linux distribution developed by NVIDIA specifically for its DGX systems and AI computing platforms. It is built on Ubuntu and optimized to run NVIDIA's AI software stack, including CUDA and cuDNN.

## Why it matters (in Poovi's context)
Ensures that Project Digits is compatible with NVIDIA's broader AI ecosystem, allowing users to run the same AI applications and libraries as on larger DGX systems.

## Key properties or components
- Linux-based
- Ubuntu foundation
- Optimized for NVIDIA hardware
- Supports NVIDIA AI stack (CUDA, cuDNN)
