## Definition
The issue where JavaScript codebases become excessively large due to the inclusion of unnecessary features, dependencies, or inefficient coding practices, leading to performance degradation.

## Why it matters (in Poovi's context)
OUI is specifically designed to combat JavaScript bloat by offering a minimal alternative.

## Key properties or components
- Large file sizes
- Excessive dependencies
- Slow loading times
- Reduced performance
