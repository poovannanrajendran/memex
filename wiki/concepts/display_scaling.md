## Definition
The process of adjusting the size of user interface elements and text on a display to improve readability or fit more content. Operating systems like macOS offer various scaling options for different resolutions.

## Why it matters (in Poovi's context)
Understanding display scaling is crucial when connecting to external monitors, as it affects the visual experience and the perceived sharpness of the UI.

## Key properties or components
- Resolution
- UI Element Size
- Readability
- External Display Compatibility
