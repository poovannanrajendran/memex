## Definition
An AI agent that assists in or automates the process of writing computer code. These agents can generate code snippets, entire functions, or even applications based on natural language descriptions or design inputs.

## Why it matters (in Poovi's context)
Kombai exemplifies the application of AI coding agents in specialised areas like frontend development, aiming to improve developer productivity.

## Key properties or components
- Code generation
- Design interpretation
- Framework support
- Automation
