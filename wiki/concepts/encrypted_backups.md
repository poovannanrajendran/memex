## Definition
A process of copying data and encrypting it before storing it, ensuring that the backed-up information is unreadable to unauthorized parties. This is crucial for data security and privacy.

## Why it matters (in Poovi's context)
Duplicati is presented as a tool that performs this function automatically, safeguarding user data when stored remotely or on external media.

## Key properties or components
- Data security
- Confidentiality
- Data integrity
- Incremental saving
