## Definition
A controlled environment specifically set up for practicing cybersecurity techniques, penetration testing, and ethical hacking. It typically involves intentionally vulnerable systems and tools for analysis.

## Why it matters (in Poovi's context)
Docker provides an efficient and isolated way to set up and tear down complex hacking labs, allowing for experimentation without risk to production systems.

## Key properties or components
- Isolated environment
- Vulnerable systems
- Security tools
- Ethical practice
