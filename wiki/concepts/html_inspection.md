## Definition
Using a web browser's developer tools to view and extract the underlying HTML source code of a webpage. This allows developers to see the structure and content of a site.

## Why it matters (in Poovi's context)
Enables the reuse of existing design structures and elements by copying HTML from inspirational websites, which can then be fed to AI for modification or refinement.

## Key properties or components
- Browser developer tools
- Element inspection
- Source code extraction
