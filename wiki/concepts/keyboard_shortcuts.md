## Definition
A set of one or more keys that invoke a command in computer software.

## Why it matters (in Poovi's context)
Significantly reduces reliance on the mouse, speeds up task execution, and improves overall navigation efficiency on a computer.

## Key properties or components
- Efficiency
- Speed
- Reduced mouse usage
- Customisation
- Accessibility
