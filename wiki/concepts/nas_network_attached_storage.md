## Definition
A dedicated file storage device that is connected to a computer network and provides storage to other devices on the network. NAS devices are commonly used for data backup, media streaming, and file sharing.

## Why it matters (in Poovi's context)
The Ooid 8 series is presented as a suitable and powerful option for building a NAS.

## Key properties or components
- Network connectivity
- Centralised storage
- File sharing
- Data backup
