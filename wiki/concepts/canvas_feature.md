## Definition
A feature within AI chat interfaces (like those from Claude, ChatGPT, and Google) that allows users to write and execute code directly within the chat environment, often limited to web technologies like HTML and JavaScript.

## Why it matters (in Poovi's context)
It provides a convenient, integrated way to experiment with code snippets and see immediate results without needing a separate development environment.

## Key properties or components
- In-browser code execution
- Support for web languages (HTML, JavaScript)
- Integration with AI chat
- Immediate feedback loop
