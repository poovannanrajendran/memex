## Definition
A collection of pre-written code, typically containing visual components and styling, designed to help developers build user interfaces more efficiently and consistently.

## Why it matters (in Poovi's context)
OUI is presented as an alternative to conventional UI libraries that are often bloated with unnecessary JavaScript.

## Key properties or components
- Reusable components
- Styling and design patterns
- JavaScript and CSS files
