## Definition
Specific software libraries, frameworks, and tools required to build, train, and deploy machine learning models.

## Why it matters (in Poovi's context)
These dependencies can be complex, platform-specific, and frequently updated, posing installation challenges.

## Key properties or components
- Libraries (e.g., TensorFlow, PyTorch)
- Compute Platforms (CPU, GPU, TPU)
- Optimized Builds (e.g., for Apple Silicon)
- CUDA/Metal Support
