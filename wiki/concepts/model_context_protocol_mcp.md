## Definition
MCP is a framework that enables large language models (LLMs) to interact with external tools and data sources, such as databases and codebases, through a secure and structured interface.

## Why it matters (in Poovi's context)
It is crucial for integrating LLMs with real-world applications and data, allowing them to perform complex tasks like stock trading analysis and execution based on live data and code.

## Key properties or components
- Framework for LLM integration
- Connects LLMs to real-world tools
- Secure and structured interface
- Enables AI-native platforms
