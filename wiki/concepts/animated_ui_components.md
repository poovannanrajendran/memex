## Definition
These are pre-designed, visually dynamic elements used in user interfaces that move or change over time. They can include buttons, menus, loaders, and other interactive parts of a digital product.

## Why it matters (in Poovi's context)
Animated UI components can enhance user experience, provide visual feedback, and make interfaces more engaging, relevant to modern web and app development.

## Key properties or components
- Dynamic visual elements
- Interactive
- Code-based implementation
- Enhance user experience
