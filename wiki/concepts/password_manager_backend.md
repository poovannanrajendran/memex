## Definition
The server-side component that stores and synchronizes encrypted password data across different devices. Users typically interact with a client application that communicates with this backend.

## Why it matters (in Poovi's context)
Vaultwarden is presented as an open-source alternative for this backend, offering users more control and potentially saving costs associated with commercial password manager services.

## Key properties or components
- Data storage
- Synchronization
- Security (encryption)
- API for clients
