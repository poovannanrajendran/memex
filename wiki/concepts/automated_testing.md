## Definition
The use of software and tools to execute pre-scripted tests on a software application before its release, verifying its functionality and performance.

## Why it matters (in Poovi's context)
Ensures software reliability and stability by catching bugs early in the development process, improving efficiency and reducing deployment risks.

## Key properties or components
- Test execution
- Coverage audits
- Performance metrics
- Regression testing
