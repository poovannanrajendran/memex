## Definition
The ability of an AI model to comprehend the full scope and relationships within data, rather than processing it in isolated parts. For coding tools, this means understanding how different files and components of a project interact.

## Why it matters (in Poovi's context)
Crucial for AI tools like GitHub Copilot to provide accurate and relevant suggestions. Graphify's innovation lies in solving this for entire codebases.

## Key properties or components
- Holistic data processing
- Inter-component relationship analysis
- Project-wide comprehension
