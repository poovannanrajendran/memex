## Definition
AI coding tools are software applications or features that leverage artificial intelligence to assist developers in writing, debugging, or optimising code. Examples include code completion, generation, and analysis tools.

## Why it matters (in Poovi's context)
These are the target tools that the agentic stack aims to integrate and enhance, standardising their operation through a common memory.

## Key properties or components
- Code generation
- Code completion
- Debugging assistance
- AI-powered analysis
