## Definition
The process of setting up an operating system on a computer. For the Raspberry Pi, this typically involves flashing an OS image onto a Micro SD card or using network boot features.

## Why it matters (in Poovi's context)
A crucial step for getting any Raspberry Pi operational, enabling the device to boot and run software applications.

## Key properties or components
- Flashing Micro SD cards
- Network installation
- Using imager tools
- First-time boot configuration
