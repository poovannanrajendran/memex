## Definition
Frameworks and platforms that enable the creation of multiple AI agents designed to collaborate and work together towards a common goal.

## Why it matters (in Poovi's context)
CrewAI is an example of an agent system that allows for the simulation of complex, multi-step reasoning by coordinating specialized AI agents.

## Key properties or components
- Multiple agents
- Defined roles and goals
- Collaborative task execution
- Orchestration
