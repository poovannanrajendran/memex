## Definition
The process by which artificial intelligence or software algorithms convert written text into graphical representations or images.

Enables users without graphic design skills to create visual content quickly, enhancing communication and engagement.

## Key properties or components

- visual generation
- customisation options
- export formats (PNG, SVG)
