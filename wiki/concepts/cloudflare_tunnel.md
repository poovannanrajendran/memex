## Definition
A Cloudflare Tunnel is a service that creates a secure, outbound-only connection from a local server or machine to Cloudflare's network. This allows Cloudflare to route external traffic to services running on the local machine without requiring the local machine to have a public IP address or open firewall ports.

## Why it matters (in Poovi's context)
Enables secure public accessibility of local AI development environments like n8n without complex network configurations.

## Key properties or components
- Secure outbound-only connection
- Bypasses firewalls
- No public IP required for the local machine
- Routable via a public domain
