## Definition
A small, reusable piece of source code that performs a specific task. In web development, these are often used for styling, animations, or functionality.

## Why it matters (in Poovi's context)
The ability to instantly copy code snippets significantly speeds up the development process by providing ready-to-use solutions for common design elements.

## Key properties or components
- Reusable
- Task-specific
- Source code
- Copyable
