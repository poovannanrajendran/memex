## Definition
A structured file or set of instructions that defines specific capabilities or functionalities for a coding agent. It allows agents to perform predefined tasks or access specific knowledge.

## Why it matters (in Poovi's context)
Enables the customization and extension of coding agents' abilities, allowing them to perform specialised tasks like nuanced CSS animation generation.

## Key properties or components
- Task definition
- Parameter specification
- Instruction sets
- Agent enablement
