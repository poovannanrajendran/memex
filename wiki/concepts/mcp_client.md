## Definition
An MCP client is responsible for managing the connections between MCP hosts and MCP servers. It facilitates the communication flow.

## Why it matters (in Poovi's context)
It acts as the intermediary, enabling the MCP host (e.g., an AI agent) to send requests to and receive responses from the MCP server.

## Key properties or components
- Manages host-server connections
- Facilitates MCP communication
