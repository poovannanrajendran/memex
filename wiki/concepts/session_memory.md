## Definition
Session memory refers to the information an AI or application retains only during the current active interaction or 'session'. Once the session concludes, this memory is typically lost.

## Why it matters (in Poovi's context)
It's the default state for many AI assistants like Claude, highlighting the need for solutions like Claude-Mem to overcome its limitations for more effective use.

## Key properties or components
- Temporary information storage
- Lost upon session termination
- Default behaviour for many AIs

## Contradictions or debates
The concept of 'endless mode' in Claude-Mem attempts to mitigate the limitations of standard session memory by compressing observations, though it's noted as experimental and not truly infinite.
