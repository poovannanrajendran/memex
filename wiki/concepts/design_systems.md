## Definition
A design system is a collection of reusable components, guided by clear standards, that can be assembled into larger applications. It aims to create a cohesive and consistent user experience across products.

## Why it matters (in Poovi's context)
Integrating design systems into AI coding tools allows for rapid generation of user interfaces that adhere to professional design principles and branding, ensuring consistency and efficiency in UI development.

## Key properties or components
- Reusable components
- Design standards
- UI consistency
- Brand alignment
