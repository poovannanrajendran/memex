## Definition
An AI operator within the context of a browser is a component that interprets user intent and page content to perform complex actions, such as filling forms, booking flights, or generating content, without requiring manual step-by-step input.

## Why it matters (in Poovi's context)
This is a core differentiator for agentic browsers like Neon, enabling seamless interaction with web pages and services, thereby reducing the need for users to engage with multiple applications or complex interfaces.

## Key properties or components
- Intent interpretation
- Page analysis
- Action execution
- Task automation
