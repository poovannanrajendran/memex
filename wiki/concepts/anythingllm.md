## Definition
AnythingLLM is an open-source desktop application that functions as an AI agent and RAG tool. It integrates with local LLMs (via Ollama) to provide advanced functionalities like web searching, memory management, and document summarization.

## Why it matters (in Poovi's context)
The central piece of software enabling local LLMs to act as sophisticated agents, offering a private and customizable alternative to cloud services.

## Key properties or components
- Local execution
- Agent orchestration
- RAG capabilities
- Integrates with Ollama
- Open source
