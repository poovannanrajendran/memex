## Definition
A display screen that is sensitive to touch, allowing users to interact with a device by touching icons or areas on the screen rather than using external input devices.

## Why it matters (in Poovi's context)
Key interface element for controlling vehicle functions in modern cars like the Range Rover, impacting user experience and design.

## Key properties or components
- Interactive display
- Replaces physical buttons
- Controls vehicle functions
- Can be curved or flat
