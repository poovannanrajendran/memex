## Definition
The fees charged by AI service providers for accessing their models and capabilities through Application Programming Interfaces (APIs). These costs are typically based on usage metrics like tokens processed or compute time.

## Why it matters (in Poovi's context)
Understanding and managing API costs is fundamental to making AI development economically viable. High API costs can quickly accumulate, necessitating strategies for reduction.

## Key properties or components
- Token pricing
- Model tiers
- Usage limits
- Provider fees
- Monthly expenditure
