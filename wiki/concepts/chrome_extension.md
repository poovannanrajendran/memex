## Definition
A Chrome extension is a small software program that customises the browsing experience. It adds extra features to the Google Chrome browser.

## Why it matters (in Poovi's context)
This is the delivery mechanism for the Chat 4 Data tool, making it easily installable and accessible within the user's existing web browser.

## Key properties or components
- Browser integration
- Feature enhancement
- User interface modification
