## Definition
An autonomous browser is a type of software that can perform web browsing tasks independently, without continuous human input. It is designed to understand user intent and execute complex actions on websites automatically.

## Why it matters (in Poovi's context)
This concept is central to Fellow AI, representing a shift from manual web navigation to automated, AI-driven research and task completion, which can drastically improve efficiency.

## Key properties or components
- Understands user intent
- Navigates websites
- Automates clicks and data collection
- Completes tasks without manual intervention
