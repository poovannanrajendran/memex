## Definition
A Plex Media Server is a piece of software that organises a user's personal media library (movies, TV shows, music, photos) and streams it to media players on various devices. It is a popular self-hosted application.

## Why it matters (in Poovi's context)
It exemplifies a common and practical use case for home servers, allowing users to create their own personal Netflix-like experience for their media collection.

## Key properties or components
- Media organisation
- Cross-device streaming
- Personal library
- Client-server model
