## Definition
A Large Language Model is a type of artificial intelligence model trained on vast amounts of text data to understand, generate, and process human language. They power applications like chatbots and text generation tools.

## Why it matters (in Poovi's context)
The core technology that Open WebUI and LiteLLM aim to provide unified access to. The ability to run both cloud-based and local LLMs offers flexibility and choice.

## Key properties or components
- Trained on massive datasets
- Generates human-like text
- Can perform various NLP tasks (summarization, translation, Q&A)
- Can be cloud-hosted or locally run
