## Definition
A type of database optimised for storing and querying vector embeddings, which are numerical representations of data (like text or images) that capture semantic meaning and relationships.

## Why it matters (in Poovi's context)
Crucial for efficient similarity search and retrieval in AI applications, particularly for implementing RAG systems and managing memory components within LLM-powered applications.

## Key properties or components
- Stores vector embeddings
- Optimised for similarity search
- Captures semantic meaning
