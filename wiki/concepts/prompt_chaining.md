## Definition
A technique where the output of one prompt or AI model is used as the input for the next prompt or model in a sequence. This allows for more complex tasks to be broken down and solved step-by-step.

## Why it matters (in Poovi's context)
Crucial for enhancing the capabilities of AI models, especially cost-effective ones like GPT-4o mini, allowing them to achieve SOTA accuracy by processing information through multiple stages.

## Key properties or components
- Sequential processing
- Input/output dependency
- Task decomposition
- Iterative refinement
