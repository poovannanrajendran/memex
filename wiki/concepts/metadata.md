## Definition
Data that provides information about other data. In web development, metadata includes information about the webpage content, such as title tags, meta descriptions, and keywords, used by search engines.

## Why it matters (in Poovi's context)
Crucial for search engine optimisation, helping search engines understand and rank the webpage content.

## Key properties or components
- Title tags
- Meta descriptions
- Keywords
- Open Graph tags

## Contradictions or debates
None.
