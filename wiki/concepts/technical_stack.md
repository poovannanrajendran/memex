## Definition
The combination of technologies, programming languages, frameworks, and tools used to build and deploy a software application.

## Why it matters (in Poovi's context)
Specifying a technical stack (Python, Cloud 3.5 Sonnet, Anthropic API) helps AI tools like ChatGPT and Coder AI propose relevant solutions and generate appropriate code.

## Key properties or components
- Programming languages
- Frameworks
- Databases
- APIs
- Cloud services
