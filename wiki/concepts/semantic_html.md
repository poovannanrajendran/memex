## Definition
The use of HTML tags according to their intended meaning (semantics) to structure web content, rather than solely for presentational purposes. This aids accessibility, SEO, and code readability.

## Why it matters (in Poovi's context)
OUI utilises semantic HTML, contributing to its efficiency and compatibility.

## Key properties or components
- Meaningful tags (e.g., <nav>, <article>)
- Improved accessibility
- Better SEO
- Enhanced code clarity
