## Definition
Meta-Command Processing (MCP) servers are an advanced feature that allows AI agents to be equipped with and utilize external tools, expanding their capabilities beyond code generation.

## Why it matters (in Poovi's context)
Enables AI agents to interact with specialized software like game engines (Unity) or data-gathering tools (Firecrawl), opening up new possibilities for AI-assisted development and research.

## Key properties or components
- Tool integration for AI
- Extends AI capabilities
- Connects to external applications
- Requires configuration
- Advanced feature
