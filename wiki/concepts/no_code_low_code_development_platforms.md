## Definition
These platforms allow users to create software applications through graphical user interfaces and configuration instead of traditional computer programming. They often use visual modelling to generate code in the background, enabling faster development cycles.

## Why it matters (in Poovi's context)
Relevant as the video showcases a platform that abstracts away complex coding, making development accessible to a wider audience, potentially impacting how businesses in insurance build internal tools or customer-facing applications.

## Key properties or components
- Visual development interface
- Pre-built components
- Drag-and-drop functionality
- Automated code generation
- Rapid prototyping
