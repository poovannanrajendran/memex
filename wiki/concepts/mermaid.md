## Definition
A JavaScript-based diagramming and charting tool that uses Markdown-inspired text definitions to create and modify diagrams dynamically.

## Why it matters (in Poovi's context)
Used as a point of comparison for Diagram Design, highlighting Diagram Design's advantages such as not requiring JavaScript or a build step.

## Key properties or components
- Markdown-like syntax
- JavaScript-based
- Diagramming tool
- Text-based definition
