## Definition
A type of artificial intelligence model trained on vast amounts of text data to understand, generate, and manipulate human language. LLMs are capable of tasks such as translation, summarisation, question answering, and text generation.

## Why it matters (in Poovi's context)
The core technology being tested in the video, with the aim of running it locally on personal hardware.

## Key properties or components
- Parameter count (e.g., 8B, 70B, 405B)
- Inference speed
- Memory requirements (RAM/VRAM)
- Computational complexity
