## Definition
Web applications that are built around a database or data model, allowing users to interact with, input, and retrieve dynamic information. The functionality is heavily dependent on the underlying data.

## Why it matters (in Poovi's context)
Power Pages enables the creation of these applications, moving beyond static websites to interactive platforms that can collect and manage business-critical information, such as investment interest.

## Key properties or components
- Database integration
- Dynamic content
- User input forms
- Data management
