## Definition
An Integrated Development Environment (IDE) enhanced with artificial intelligence capabilities, specifically designed to work with AI coding assistants. These IDEs facilitate seamless interaction between the developer, the code, and the AI.

## Why it matters (in Poovi's context)
Provides a specialized environment for leveraging AI coding assistants effectively.

## Key properties or components
- Code editor
- Debugging tools
- AI integration
- Plugin support
- Context management
