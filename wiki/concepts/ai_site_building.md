## Definition
The process of using artificial intelligence tools, such as AI models, to assist in or fully generate the code and design elements required for building websites.

## Why it matters (in Poovi's context)
Enables faster and potentially more efficient website creation, particularly for generating premium-looking designs with less manual effort.

## Key properties or components
- Leverages AI models for code generation
- Can incorporate specific design instructions and assets
- Aims to reduce development time and complexity
