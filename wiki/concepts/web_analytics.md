## Definition
Web analytics is the measurement, collection, analysis, and reporting of web data for purposes of understanding and optimizing web usage.

## Why it matters (in Poovi's context)
Provides critical insights into user behaviour on a website, helping businesses identify issues, understand customer journeys, and make data-driven decisions to improve performance and conversion rates.

## Key properties or components
- Tracks user interactions
- Identifies drop-off points
- Measures traffic sources
- Informs website optimization
- Tools include heatmaps and session recordings
