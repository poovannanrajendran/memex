## Definition
These are components or functionalities within the Claude Code framework. Sub-agents suggest modular AI entities, skills represent specific capabilities, commands are direct instructions, and hooks are mechanisms for triggering automated actions, such as code formatting.

## Why it matters (in Poovi's context)
Understanding the distinctions and appropriate use cases for each of these components is vital for building effective and efficient agentic workflows in Claude Code. The repository clarifies these differences.

## Key properties or components
- Modularity (Sub-agents)
- Capabilities (Skills)
- Instructions (Commands)
- Automation triggers (Hooks)
