## Definition
Multi-column layouts in Notion allow users to arrange content blocks side-by-side on a page, creating a more organized and visually appealing structure. This is achieved by dragging blocks next to each other until a visual guide appears.

## Why it matters (in Poovi's context)
Enables more sophisticated page design and information presentation, allowing users to break up text, display related information adjacently, and create more magazine-like layouts.

## Key properties or components
- Side-by-side arrangement of blocks
- Adjustable column widths
- Requires full-width page for more than two columns
- Can contain nested layouts
