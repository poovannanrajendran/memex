## Definition
Software applications designed to control drones for specific data collection tasks, such as aerial photography, surveying, mapping, and inspection. These apps often feature automated flight planning, real-time data monitoring, and integration with processing software.

## Why it matters (in Poovi's context)
Essential tools for professionals and hobbyists looking to leverage drones for data acquisition, enabling efficient and precise capture of aerial imagery and sensor data for various applications.

## Key properties or components
- Automated flight planning
- Real-time monitoring
- Geotagging
- Data export
- Compatibility with drone hardware
