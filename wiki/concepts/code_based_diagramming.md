## Definition
The method of creating diagrams by writing code or text-based descriptions, which are then rendered into visual diagrams, rather than using traditional graphical user interfaces.

## Why it matters (in Poovi's context)
This approach is important for version control, automation, and ensuring consistency in diagram creation. Its integration with AI, as seen with Claude, allows for programmatic and dynamic generation of complex visuals, which is beneficial for AI development and software engineering.

## Key properties or components
- Text or code as input
- Generates visual diagrams
- Enables automation and version control
- Often integrates into development workflows
