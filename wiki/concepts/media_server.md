## Definition
A computer or device that stores and serves media content, such as videos, music, and photos, to other devices on a network. This allows for centralised media libraries accessible from various playback devices.

## Why it matters (in Poovi's context)
The Ooid 8 series is highlighted as a capable platform for creating a personal media server.

## Key properties or components
- Media storage
- Network streaming
- Content organisation
- Remote access
