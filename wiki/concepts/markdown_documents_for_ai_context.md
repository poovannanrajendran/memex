## Definition
Using markdown-formatted files (e.g., .md) to provide structured context, instructions, and documentation to AI coding assistants. These documents serve as a high-level reference for planning, architecture, and tasks.

## Why it matters (in Poovi's context)
Enhances AI's understanding of project scope and requirements, leading to more relevant code generation.

## Key properties or components
- Structured text
- Hierarchical organisation
- Readability
- Version control friendly
