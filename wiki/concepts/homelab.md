## Definition
A personal computer network or server setup maintained at home for experimental, educational, or self-hosting purposes. It often involves using open-source software like Proxmox for virtualisation to run various services.

## Why it matters (in Poovi's context)
The author uses a 'Proxmox homelab' to run a local agent via OpenClaw, indicating a preference for self-hosting and local control over certain components of their memex system, aligning with a hands-on approach.

## Key properties or components
- personal
- self-built
- for learning/experimentation
- involves servers/networking/virtualisation
- self-hosting
