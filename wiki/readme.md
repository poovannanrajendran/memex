# memex

A personal knowledge base maintained by a Gemini CLI agent,
built on Andrej Karpathy's LLM Wiki pattern.

Raw sources go in → the agent compiles a living, interlinked wiki → browse in Obsidian.

**Stack:** Gemini CLI · Gemini 2.5 Pro/Flash · Obsidian · Python · Git
