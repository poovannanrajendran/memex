## Summary
Open Montage is a free, agentic video production system that transforms plain English descriptions into complete videos. It leverages AI to handle research, scripting, voice-over, visuals, and editing, utilising a vast array of tools and skills. The system grounds video content in real-time web searches and can produce various video types, integrating with tools like Claude code, Cursor, and Copilot without requiring API keys.

## Key claims
- Open Montage is the world's first agentic video production system.
- It can produce various video types, from animated explainers to cinematic trailers and product ads.
- The system conducts live web searches for research before scripting.
- Open Montage operates using free tools locally, without needing API keys to start.
- A full cinematic sci-fi trailer and product ad were produced for a total cost of 69 cents.
