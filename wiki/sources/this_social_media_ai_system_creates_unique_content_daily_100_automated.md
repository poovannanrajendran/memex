## Summary
This video demonstrates how to set up an automated social media content creation system using Make.com. The system is triggered by new articles added to a Google Sheet. It then uses Perplexity.ai to summarise the article, and subsequently employs Claude and OpenAI to generate unique posts for Facebook, Instagram, X (Twitter), and LinkedIn. The process is entirely no-code and allows for customisation of prompts to achieve desired output and tone.

## Key claims
- An AI system can be automated to create and post unique social media content daily.
- Make.com is a no-code RPA tool suitable for building such automation workflows.
- Perplexity.ai can access the web in real-time to summarise articles, outperforming models with older data.
- Claude and OpenAI can be used to generate engaging social media posts tailored for different platforms.
- The entire automation system can be set up within minutes without writing any code.
