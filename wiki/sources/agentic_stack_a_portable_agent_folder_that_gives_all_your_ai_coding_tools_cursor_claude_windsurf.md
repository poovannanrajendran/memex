## Summary
This video introduces "Agentic Stack", a solution to the common problem of fragmented memory across various AI coding tools like Cursor, Claude code, and WinSurf. It proposes a portable ".agent" folder designed to centralise an AI's memory, custom skills, and communication protocols. This standardisation allows for a unified "brain" that can be plugged into diverse AI environments, improving efficiency and consistency in AI-assisted development.

## Key claims
- Existing AI coding tools like Cursor, Claude code, and WinSurf suffer from isolated memory, meaning rules taught to one are not remembered by others.
- Agentic Stack solves this by providing a portable ".agent" folder that serves as a single brain for multiple AI harnesses.
- The ".agent" folder standardises AI's memory, custom skills, and protocols.
- It can be integrated with various AI environments, including Claude code, Cursor, WinSurf, Open Claw, or custom Python loops.
