## Summary
This YouTube video introduces three free tools that enhance the capabilities of ClaudeCode, a powerful AI coding agent. These tools, Vibe Kanban, Get Done, and ClaudeCode Templates, aim to increase productivity by allowing users to manage multiple AI agents simultaneously, generate and execute comprehensive project plans, and utilise pre-built agent setups, respectively. The video emphasizes that these tools provide crucial control and efficiency, enabling users to build projects significantly faster than using ClaudeCode alone.

## Key claims
- Using ClaudeCode without these three free tools makes users work ten times harder.
- Vibe Kanban allows running multiple AI coding agents simultaneously, akin to managing a development team.
- Get Done solves ClaudeCode's tendency to lose track of tasks mid-project by creating and executing a plan.
- ClaudeCode Templates offer over 100 pre-built agent setups, saving significant configuration time.
- All three tools are free and dramatically accelerate project development.
