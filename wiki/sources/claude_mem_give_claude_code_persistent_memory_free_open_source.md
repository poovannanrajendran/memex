## Summary
Claude-Mem is a new, free, and open-source plugin that provides persistent memory to the Claude code AI assistant. It allows Claude to remember workflows, decisions, and tool outputs across sessions, overcoming its usual limitation of forgetting everything when a session ends. The plugin is available via the Claude code plugin marketplace and includes features for searching past memories and an experimental 'endless mode' for longer sessions.

## Key claims
- Claude-Mem adds persistent memory to Claude code, enabling it to remember information across sessions.
- The plugin stores key observations and retrieves them in future sessions.
- Claude-Mem is 100% open-source and installable through the Claude code plugin marketplace.
- It offers a local memory viewer and a skill-based retrieval system for searching past memories.
- An experimental 'endless mode' aims to compress observations for longer sessions.
