## Summary
Opera has launched Neon, an "agentic browser" designed for the new AI-driven web. This browser integrates several AI features directly, such as instant research and content generation (similar to Perplexity), page understanding for actions like booking flights or filling forms, and the ability to create web applications through simple prompts. The platform aims to consolidate these functionalities, potentially rendering numerous AI startups obsolete by making their core offerings mere features within the browser.

## Key claims
- Opera's new browser, Neon, is the first "agentic browser" for the new web.
- Neon integrates AI features like instant research, content generation, and page interaction, making standalone AI tools like Perplexity redundant as features.
- Neon allows users to build applications by interpreting prompts, challenging AI builders like Replate and Bold.
- The introduction of Neon could make many existing AI startups worthless due to its integrated functionalities.
