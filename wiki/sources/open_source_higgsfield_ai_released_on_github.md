## Summary
A developer has created and released a free, open-source version of Higgsfield AI on GitHub. This version provides a single interface to over 200 AI models for various creative workflows, including image, video, and lip sync. While the software is free, users must provide their own API keys for the background processing, meaning they pay only for their usage of the underlying AI models.

## Key claims
- An open-source version of Higgsfield AI has been released for free on GitHub.
- The open-source version offers access to over 200 AI models for diverse creative tasks.
- The core functionality, particularly the 'cinema studio' feature, mirrors Higgsfield's paid offering by translating camera settings into structured prompts for AI generation.
- Users of the open-source version must supply their own API keys, incurring costs based on their usage of background processing.
- The open-source alternative has gained significant traction, with over 5,000 GitHub stars in a few days, indicating strong competition for commercial AI video tools.
