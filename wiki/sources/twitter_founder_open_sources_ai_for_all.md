## Summary
Jack Dorsey's company, Block, has launched Goose, a fully open-source AI agent designed to assist developers. Unlike paid alternatives, Goose is free and integrates with over 70 MCP servers, allowing it to work across an entire tech stack and with any LLM. This initiative aims to make AI development tools accessible to everyone, with the belief that broader adoption will strengthen the tool.

## Key claims
- Developer tools are becoming increasingly expensive.
- Goose is a free, open-source AI agent developed by Block.
- Goose can install, execute, edit, and test code on a user's machine.
- Goose connects to over 70 MCP servers, enabling cross-stack functionality.
- Goose is compatible with any Large Language Model (LLM), including cloud-based and local options.
- Block's engineers report time savings of up to 20% using Goose.
- The open-source nature of Goose is intended to foster community development and widespread AI accessibility.
- The project has gained significant traction on GitHub with over 39,000 stars.
