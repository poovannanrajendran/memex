## Summary
This video explains how to modify ChatGPT's output to be more direct and less flattering. It suggests using custom instructions to prompt ChatGPT to avoid excessive praise, push back on negative ideas, and provide constructive feedback. Additionally, it recommends instructions for clearer communication, such as writing directly, avoiding jargon, using plain language, and employing formatting like bullet points and tables.

## Key claims
- ChatGPT is often overly flattering due to its default settings.
- Custom instructions can be used to change ChatGPT's personality to be more direct and critical.
- Specific custom instructions can improve ChatGPT's output by guiding its tone, feedback style, and writing conventions.
