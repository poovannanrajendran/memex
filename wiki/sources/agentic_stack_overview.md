## Summary
The agentic-stack is a concept and toolset designed to solve the problem of fragmented memory across different AI coding tools (Cursor, Claude Code, Windsurf). It proposes a portable `.agent` folder that standardises memory, skills, and protocols, acting as a single "brain" that can be shared across multiple "harnesses" (tools).

## Key claims
- Different AI tools have isolated memory, leading to inconsistent behavior.
- Agentic-stack provides a portable `.agent` folder for standardisation.
- It enables "One brain, many harnesses."
- Plugs into Claude Code, Cursor, Windsurf, OpenClaw, and custom Python loops.
