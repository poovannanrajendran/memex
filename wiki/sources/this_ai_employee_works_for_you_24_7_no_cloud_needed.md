## Summary
A new free AI agent named Deerflow has been released from China that operates entirely locally, without requiring cloud access. This agent can perform a variety of tasks including research, coding, creating slide decks, and generating videos autonomously. It functions as a super agent, breaking down large tasks into smaller steps and assembling them into a final product, with the ability to interact with files, browse the web, and save outputs directly to the user's machine, all while learning user preferences. This functionality is comparable to paid tools like Manis and Perplexity.

## Key claims
- Deerflow is a free AI agent that runs 100% locally.
- Deerflow can perform research, code, create slide decks, and generate videos autonomously.
- Deerflow acts as a super agent, breaking down big tasks into smaller pieces and assembling them into a finished output.
- Deerflow has its own sandbox computer allowing it to read/write files and browse the web.
- Deerflow saves results directly to the user's machine and remembers how the user works.
- Deerflow offers functionality similar to paid tools like Manis and Perplexity for free.
