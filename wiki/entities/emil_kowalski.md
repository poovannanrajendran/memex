## Overview
Emil Kowalski is a design engineer with a background at Vercel, where he created open-source animation libraries. He is now focused on developing AI-powered tools for web development.

## Role in this knowledge base
Creator of open-source animation libraries and a skill file for coding agents, aiming to enhance web animation creation.

## Key facts
- He built Sauner and Ball, two open-source libraries with over 20,000 GitHub stars.
