## Overview
A modern, extensible productivity launcher for macOS that enhances system navigation and task execution through a command-line interface.

## Role in this knowledge base
Recommended as an alternative or successor to Alfred, offering a more intuitive interface and powerful features for launching apps, searching files, and running custom commands.

## Key facts
- It replaces the default Spotlight search and allows users to assign specific hotkeys to open applications or execute commands.
