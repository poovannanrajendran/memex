## Overview
The Snapdragon XR2 Gen 2 is a chipset designed by Qualcomm specifically for virtual reality and augmented reality devices. It provides enhanced performance for XR applications.

## Role in this knowledge base
The processor powering both the Quest 3 and Quest 3S, indicating shared core performance capabilities.

## Key facts
- It is the same chipset used in both the Quest 3 and Quest 3S.
