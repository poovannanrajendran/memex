## Overview
The Radeon 780M is an integrated GPU (iGPU) based on AMD's RDNA 3 architecture, found in some Ryzen processors. It offers improved graphics performance for integrated solutions.

## Role in this knowledge base
This iGPU was tested for its potential to accelerate LLM inference on the Herk Mini PC, but it encountered compatibility issues with Ollama.

## Key facts
- It was marketed as an AI Mini PC GPU but Ollama's server logs indicated no compatible GPUs were discovered, suggesting it's not supported for LLM acceleration by this software.
