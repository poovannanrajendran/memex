## Overview
Plane is a minimalistic and free open-source project management tool. It offers core collaboration features without unnecessary complexity and can be self-hosted. An upgraded version with additional features is available as a lifetime deal.

## Role in this knowledge base
Replaces the project management function previously handled by Notion.

## Key facts
- It offers a $799 lifetime deal for additional features like Cycles, time tracking, and real-time collaboration.
