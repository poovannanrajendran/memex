## Overview
GitHub is a web-based platform offering distributed version control and source code management using Git. It provides tools for collaboration, issue tracking, and code hosting.

## Role in this knowledge base
Serves as a cloud-based storage solution for Git repositories, offering secure code backup and collaboration features.

## Key facts
- GitHub is described as 'like Google Docs for code' and is free to sign up for.
