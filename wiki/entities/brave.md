## Overview
Brave is a web browser with built-in search capabilities.

## Role in this knowledge base
It serves as the web search MCP server, providing AI coding assistants with access to up-to-date information from the internet.

## Key facts
- It can be leveraged by AI coding assistants if they need to search for additional resources.
