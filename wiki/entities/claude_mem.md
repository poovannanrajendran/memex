## Overview
Claude-Mem is a free and open-source plugin designed to add persistent memory capabilities to the Claude code AI.

## Role in this knowledge base
The tool that provides persistent memory to Claude, enhancing its ability to learn and retain information over time.

## Key facts
- It stores key observations and retrieves them in future sessions.
