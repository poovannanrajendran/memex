## Overview
Visual Studio Code (VS Code) is a widely popular, free source-code editor developed by Microsoft, known for its extensive extension ecosystem and customisability.

## Role in this knowledge base
Serves as the foundation for tools like Windsurf and is mentioned as an environment where extensions like Klein can be used for AI coding assistance.

## Key facts
- VS Code is described as the 'most popular coding editor on the planet.'
