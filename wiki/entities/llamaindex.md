## Overview
An open-source data orchestration framework that has introduced 'Workflows' based on an event-driven architecture, featuring steps, events, and a shared context.

## Role in this knowledge base
Presented as an AI agent framework offering high flexibility for dynamic applications that require frequent looping or branching behaviours.

## Key facts
- LlamaIndex's 'Workflows' components communicate via events rather than predefined paths.
