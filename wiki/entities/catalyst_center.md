## Overview
Catalyst Center is Cisco's network management solution that allows for the orchestration, automation, and assurance of the network. It provides a unified platform for managing network infrastructure.

## Role in this knowledge base
It is mentioned as a platform for managing Cisco's new access points, either on-premise or as part of a hybrid approach.

## Key facts
- Catalyst Center can be used to manage Cisco access points on-premise.
