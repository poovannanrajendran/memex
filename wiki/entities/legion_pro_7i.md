## Overview
The Legion Pro 7i is a powerful Windows/Linux laptop featuring a high-end Intel processor and an Nvidia RTX 4090 GPU option with 16GB of memory. It offers excellent performance due to robust cooling and upgradeable memory slots.

## Role in this knowledge base
A top-tier choice for users needing extreme performance for ML/AI on Windows or Linux, particularly those who require Nvidia's CUDA support and a high-performance GPU.

## Key facts
- It can be equipped with up to an Nvidia RTX 4090 with 16GB memory and has two upgradeable memory slots for up to 64GB of system memory.
