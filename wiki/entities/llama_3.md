## Overview
Llama 3 is a large language model developed by Meta AI, known for its strong performance and open availability for research and commercial use.

## Role in this knowledge base
A locally hosted LLM that can be run alongside cloud-based models within Open WebUI.

## Key facts
- Open WebUI supports running local LLMs like Llama 3, which can be accessed directly from the interface without requiring API keys or external services.
