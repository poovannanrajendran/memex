## Overview
Canvas is a feature within ChatGPT that provides a split-view interface with a chat window on the left and a document editor on the right, facilitating collaborative writing and coding.

## Role in this knowledge base
Presented as a key feature for smoother writing and coding experiences within ChatGPT.

## Key facts
- Canvas offers real-time editing suggestions, adjustable reading levels, and grammar checks directly within the document editor.
