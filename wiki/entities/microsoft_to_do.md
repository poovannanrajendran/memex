## Overview
Microsoft To Do is a task management application that syncs with Microsoft Outlook tasks and offers a simple interface for managing daily to-do lists.

## Role in this knowledge base
Provides a consolidated view of all tasks assigned to a user across different projects and platforms, accessible on mobile devices.

## Key facts
- Automatically syncs and consolidates tasks from Outlook and Planner into a single 'Tasks assigned to me' list.
