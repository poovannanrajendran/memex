## Overview
Docker Desktop is an application for macOS and Windows that makes it easy to build and share containerized applications. It includes Docker Engine, Docker CLI client, Docker Compose, and Kubernetes.

## Role in this knowledge base
Presented as a user-friendly graphical interface for managing Docker containers, images, and settings, including extensions.

## Key facts
- Docker Desktop provides a graphical user interface for managing Docker environments and supports extensions to add functionality.
