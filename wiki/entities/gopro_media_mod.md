## Overview
The GoPro Media Mod is an accessory that expands the functionality of GoPro cameras, adding features like external microphone input and improved built-in microphones.

## Role in this knowledge base
An accessory that enhances the audio and connectivity options for the GoPro Hero 13 Black.

## Key facts
- It allows for the connection of an external 3.5mm microphone to the GoPro Hero 13 Black.
