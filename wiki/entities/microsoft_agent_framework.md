## Overview
A development framework for building AI applications, combining features from AutoGen and Semantic Kernel. It provides tools for creating individual AI agents and orchestrating them using graph-based workflows.

## Role in this knowledge base
The primary subject of the article, representing a new approach to AI development.

## Key facts
- It is the direct successor to both Microsoft's AutoGen and Semantic Kernel projects.
