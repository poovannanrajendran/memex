## Overview
emergent.sh is a no-code platform that uses AI to generate 3D animated websites based on user input. It allows users to select pre-made animated icons and provide a text prompt to define the website's style and content.

## Role in this knowledge base
The core tool demonstrated in the video for creating 3D websites.

## Key facts
- It can generate a complete, animated website from a single prompt and a selection of icons.
