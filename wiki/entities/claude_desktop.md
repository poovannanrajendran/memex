## Overview
Claude Desktop is a desktop application that provides access to Anthropic's Claude language models.

## Role in this knowledge base
It is presented as an example of an MCP host that can be configured to communicate with MCP servers, such as those built in n8n.

## Key facts
- Claude Desktop needs to be in developer mode and configured with specific JSON settings to communicate with MCP servers via a gateway.
