## Overview
An open-source, self-hostable implementation of the Tailscale control server. It allows users to run their own Tailscale network without relying on Tailscale's cloud infrastructure.

## Role in this knowledge base
Presented as an alternative to Tailscale for those who want to self-host their control plane, though noted for the lack of readily available apps compared to Tailscale.

## Key facts
- It allows users to self-host the control plane for a Tailscale-like network.
