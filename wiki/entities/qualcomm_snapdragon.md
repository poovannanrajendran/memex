## Overview
Qualcomm Snapdragon processors are a series of system on a chip (SoC) found in many mobile devices and some laptops, often utilising the ARM architecture. Laptops with these processors run an ARM version of Windows.

## Role in this knowledge base
Flagged as a caution for data scientists, as several key data science programs (like Matlab) do not run natively or with full performance on Windows ARM due to compatibility issues, even with emulation.

## Key facts
- Data science programs like Tableau and Power BI can run under emulation on Qualcomm Snapdragon processors using the ARM version of Windows, but Matlab does not.
