## Overview
Get Done is an AI tool that takes a user's desired outcome, creates a detailed plan with actionable steps, and then executes that plan to build the project. Its key feature is its ability to maintain context and complete projects without forgetting steps.

## Role in this knowledge base
Addresses a common limitation of AI coding tools like ClaudeCode by ensuring project completion through robust planning and execution, preventing mid-project drop-offs.

## Key facts
- It creates a plan and breaks it into steps, then builds the project without losing track halfway through.
