## Overview
Playwright is a browser automation tool that allows for programmatic control of web browsers. The CLI (Command Line Interface) version enables scripting and automation of browser-based tasks.

## Role in this knowledge base
Used to automate the process of logging into Higgsfield, selecting Seed Dance, pasting prompts, and uploading reference images, streamlining the AI-driven video creation workflow.

## Key facts
- It automates the entire prompt generation and submission process for Higgsfield and Seed Dance.
