## Overview
The Mac Pro is Apple's high-end professional desktop workstation. The M2 Ultra model features a powerful system-on-a-chip (SoC) with integrated CPU, GPU, and unified memory.

## Role in this knowledge base
This machine was tested to assess LLM performance leveraging its unified memory architecture, which allows the GPU to access a large pool of system RAM.

## Key facts
- It features an M2 Ultra chip with 128GB of unified memory, which is all available to the GPU.
