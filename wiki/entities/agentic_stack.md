## Overview
The agentic-stack is an architecture for creating portable, interoperable AI agent configurations. It uses a standard `.agent` directory to store memory, skills, and instructions.

## Role in this knowledge base
Represents a solution to the "amnesia" problem when switching between development tools like [[claude_code]] and [[cursor]]. It ensures Poovi's custom rules and knowledge are available to all his agents.

## Key facts
- Concept: "One brain, many harnesses."
- Compatible with [[openclaw]].
