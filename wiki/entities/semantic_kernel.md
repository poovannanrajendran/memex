## Overview
An open-source SDK from Microsoft for building enterprise-grade Generative AI applications, providing abstractions for chat completion and assistant agents.

## Role in this knowledge base
Functions as both an agent framework and a process framework, enabling the orchestration of multiple agents through group chats or complex data-flow processes.

## Key facts
- Semantic Kernel facilitates the building of enterprise-grade GenAI applications through its agent and process frameworks.
