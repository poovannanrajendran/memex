## Overview
NVIDIA DGX OS is a Linux-based operating system specifically designed for NVIDIA's DGX AI systems. It provides an optimized environment for running AI workloads and leveraging NVIDIA's hardware.

## Role in this knowledge base
The operating system that Project Digits runs on, ensuring compatibility with NVIDIA's AI software stack.

## Key facts
- Project Digits runs on DGX OS Linux, which is based on Ubuntu, making it compatible with NVIDIA's full AI stack.
