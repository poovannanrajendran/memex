A free, lightweight, yet powerful source code editor developed by Microsoft, supporting a vast array of programming languages and extensions.

## Role in this knowledge base
A frequently used editor for development tasks, emphasized by the setup of its command-line integration.

## Key facts
- The 'code' command can be installed to launch VS Code directly from the terminal.
