## Overview
AnythingLLM is an open-source, all-in-one AI agent and RAG (Retrieval Augmented Generation) tool that runs locally on a user's desktop. It aims to provide enhanced capabilities to local LLMs.

## Role in this knowledge base
The primary tool used in the video to enable agent capabilities for local LLMs running on Ollama.

## Key facts
- AnythingLLM ships with its own embedder and vector database for local processing.
