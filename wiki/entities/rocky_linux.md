## Overview
Rocky Linux is a community-driven, enterprise-class Linux distribution that serves as a free, downstream rebuild of Red Hat Enterprise Linux (RHEL). It aims to be a stable and reliable operating system.

## Role in this knowledge base
Shown as an example of how to easily spin up and test different operating systems like Rocky Linux within Docker containers.

## Key facts
- Various Linux distributions, including Rocky Linux, can be easily run and tested using Docker.
