## Overview
Kali Linux is a Debian-based Linux distribution designed for digital forensics and penetration testing. It comes pre-installed with a vast array of security and penetration testing tools.

## Role in this knowledge base
Demonstrated as a powerful security tool that can be run within a Docker container, ideal for setting up isolated hacking labs.

## Key facts
- Kali Linux can be run as a GUI application within a Docker container, accessible via a web browser.
