## Overview
An open-source framework developed by Microsoft for creating multi-agent applications, featuring a scalable, distributed network of agents with asynchronous messaging.

## Role in this knowledge base
Presented as a popular AI agent framework capable of building multi-agent systems, suitable for various applications.

## Key facts
- AutoGen includes AgentChat for conversational teams and AutoGen Studio for a no-code interface.
