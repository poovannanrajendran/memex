## Overview
The Brave Search API allows developers to programmatically access Brave's web search capabilities, providing search results for various queries.

## Role in this knowledge base
Used as the primary tool in the video's demonstration of building a web search agent with Pantic AI, showcasing how agents can interact with real-world data.

## Key facts
- It is a free-to-use API that enables agents to perform web searches.
