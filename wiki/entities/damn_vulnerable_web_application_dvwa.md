## Overview
Damn Vulnerable Web Application (DVWA) is a PHP/MySQL web application that is intentionally vulnerable. It is used for security education and training purposes to practice penetration testing.

## Role in this knowledge base
Used as an example of a vulnerable machine that can be deployed as a Docker container within an isolated network for ethical hacking practice.

## Key facts
- DVWA can be deployed as a Docker container within a specifically created isolated Docker network for security practice.
