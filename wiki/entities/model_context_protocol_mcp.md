## Overview
MCP is a framework that enables large language models (LLMs) to connect to external tools like databases, codebases, and file systems through a secure and structured interface. It facilitates interaction between AI models and the real world.

## Role in this knowledge base
MCP is the core technology enabling the AI agents to interact with data and code, turning the trading application into an AI-native platform.

## Key facts
- MCP allows LLMs to connect to real-world tools such as databases, codebases, or file systems using a secure and structured interface.
