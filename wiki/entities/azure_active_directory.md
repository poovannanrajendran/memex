## Overview
Azure Active Directory (Azure AD) is Microsoft's cloud-based identity and access management service. It helps employees sign in and access resources, such as thousands of other SaaS applications.

## Role in this knowledge base
Mentioned as an identity provider that can be used with Power Pages to set up user logins for websites.

## Key facts
- Azure Active Directory can be configured with Power Pages for website identity management.
