## Overview
Hermes Agent is an open-source AI agent framework that focuses on creating and refining skills from experience, alongside building a persistent user model across sessions. It aims to create agents that improve their performance over time through a structured learning loop.

## Role in this knowledge base
The primary subject of the article, presented as an alternative to other agent frameworks like OpenClaw, with a focus on its unique learning capabilities.

## Key facts
- It is an open-source AI agent framework that creates, refines, and retrieves skills from experience, and builds a persistent user model.
