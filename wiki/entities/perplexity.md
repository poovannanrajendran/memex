## Overview
Perplexity is an AI-powered conversational search engine that provides real-time web search capabilities. It can find and summarise sources, and users can create 'Spaces' to organize and search within specific sets of information.

## Role in this knowledge base
Perplexity acts as the primary source-gathering tool, identifying and collecting relevant articles, reports, studies, and user feedback from the open web based on user prompts.

## Key facts
- Perplexity's limitation is that each question triggers a new search, potentially using different sets of sources.
