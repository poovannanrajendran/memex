## Overview
Super Gateway is presented as a tool that acts as a bridge, enabling communication between MCP hosts and MCP triggers that use SSE (Server-Sent Events) protocols.

## Role in this knowledge base
It is used as a dependency to allow Claude Desktop to communicate with n8n's MCP trigger, which utilizes SSE.

## Key facts
- Super Gateway is used to bridge the communication gap for SSE protocols between Claude Desktop and n8n's MCP trigger.
