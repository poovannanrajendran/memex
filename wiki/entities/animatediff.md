## Overview
AnimateDiff is an open-source motion generation model that can be integrated with Stable Diffusion and ComfyUI to create animated sequences from image models.

## Role in this knowledge base
A key open-source component used with tools like ComfyUI to enable advanced AI video generation.

## Key facts
- AnimateDiff is crucial for creating animated video sequences within open-source frameworks like ComfyUI, offering deep customisation.
