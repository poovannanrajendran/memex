## Overview
Open WebUI is an open-source, self-hosted web interface for interacting with various Large Language Models (LLMs). It provides a unified platform to access and manage different AI models, both cloud-based and locally hosted.

## Role in this knowledge base
The primary interface for accessing and managing multiple AI models in the described setup.

## Key facts
- It offers a user-friendly interface that allows for the creation of user accounts, customisation of access, and integration with other AI services via APIs.
