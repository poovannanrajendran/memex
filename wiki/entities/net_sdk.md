## Overview
A collection of tools and libraries for building and running .NET applications, supporting cross-platform development with languages like C#.

## Role in this knowledge base
Used for server-side development and cross-platform application building.

## Key facts
- Version 6 is available as Long-Term Support (LTS), and it can be installed directly or via Visual Studio for Mac.
