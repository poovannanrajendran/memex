## Overview
A free, open-source, cross-platform, lightweight, high-performance, production-ready distribution of the OpenJDK.

## Role in this knowledge base
Recommended for Java Development Kit (JDK) installation, supporting Apple Silicon.

## Key facts
- It can be installed via Homebrew using the command `brew install --cask temurin`.
