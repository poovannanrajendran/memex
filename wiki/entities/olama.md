## Overview
Olama is a tool that simplifies the process of running large language models (LLMs) locally. It provides an easy-to-use interface for downloading, managing, and interacting with various open-source LLMs.

## Role in this knowledge base
One of the self-hosted AI starter kit components running in the Docker container.

## Key facts
- Olama is part of the self-hosted AI starter kit along with n8n, Postgress, and Quadrant.
