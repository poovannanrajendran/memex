## Overview
OneNote is a digital note-taking application that allows users to create notes, drawings, screen clippings, and audio commentaries.

## Role in this knowledge base
Can be integrated with Outlook tasks by flagging notes, which then generates a task with a deadline in Outlook.

## Key facts
- Users can right-click a OneNote note and select a flag option to convert it into an Outlook task.
