## Overview
Hyperframes is an open-sourced framework developed by HeyGen that converts HTML code into MP4 videos. It is designed to be used by AI agents and can create videos quickly and cost-effectively.

## Role in this knowledge base
The core AI tool discussed for generating videos from HTML.

## Key facts
- It is open-sourced under the Apache 2 license.
