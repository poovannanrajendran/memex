## Overview
Power Automate (formerly Microsoft Flow) is a cloud-based service that allows users to create and automate workflows across multiple applications and services.

## Role in this knowledge base
Highlighted as a tool that can be used with Power Pages to automate processes, such as sending emails or posting messages when a form is submitted.

## Key facts
- Power Automate can be used to automate flows triggered by form submissions on a Power Pages website.
