## Overview
Wide Bundles is an application for Shopify that focuses on creating product bundles, allowing merchants to offer package deals and increase average order value.

## Role in this knowledge base
This app is recommended as a tool for implementing bundling strategies, which are similar to upsells but specifically designed for selling multiple units of the same or complementary products.

## Key facts
- It helps in creating package deals like 'buy one get one' or multi-packs.
