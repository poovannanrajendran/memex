## Overview
Hotjar is a web analytics and user feedback service that provides insights into user behaviour through tools like heatmaps, session recordings, and surveys.

## Role in this knowledge base
Hotjar is recommended for analysing customer interactions on a Shopify store, enabling businesses to understand user drop-off points and optimize the website for better conversion rates.

## Key facts
- It records every single user interaction on a website, showing mouse movements, clicks, and drop-off points.
