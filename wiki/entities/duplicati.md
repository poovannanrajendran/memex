## Overview
Duplicati is a free, open-source backup client that securely stores incremental backups of data on cloud storage services. It supports encryption and various back-end storage options.

## Role in this knowledge base
Recommended as an essential tool for creating automatic, encrypted backups of files to cloud storage or external drives.

## Key facts
- Creates automatic, encrypted backups of files to cloud storage services.
