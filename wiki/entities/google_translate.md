## Overview
Google Translate is a free multilingual neural machine translation service developed by Google. It provides instant translations of words, phrases, and web pages between numerous languages.

## Role in this knowledge base
Facilitates communication with locals through voice and text translation, and offers real-time visual translation using a phone's camera.

## Key facts
- Users can download languages for offline use, and it supports features like camera translation and handwriting input.
