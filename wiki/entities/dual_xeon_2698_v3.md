## Overview
The Intel Xeon E5-2698 v3 is a server/workstation processor from the Haswell-EP generation, featuring 16 cores and 32 threads per CPU. A dual-socket configuration would offer a substantial core count for demanding tasks.

## Role in this knowledge base
Part of the creator's initial, more expensive build idea, which involved a custom water-cooled dual Xeon system.
