## Overview
Raspberry Pi OS (formerly Raspbian) is the official operating system for Raspberry Pi devices, based on Debian Linux. It provides a user-friendly desktop environment and includes essential software for development and general computing.

## Role in this knowledge base
The primary operating system installed and used on the Raspberry Pi 5 for the projects.

## Key facts
- Raspberry Pi OS is a custom-skinned version of Linux that includes a menu bar, web browser, file manager, and terminal.
