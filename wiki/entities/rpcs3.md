## Overview
RPCS3 is a free and open-source in-development multi-platform Sony PlayStation 3 emulator and debugger written in C++ for Windows and Linux. It aims to accurately emulate the PS3's complex hardware and software stack.

## Role in this knowledge base
A tool used to test the emulation capabilities of the Jetson Orin Nano SUPER, showcasing its ability to run PS3 games on an ARM device.

## Key facts
- A native ARM version of RPCS3 is available and functions on the Orin Nano SUPER.
