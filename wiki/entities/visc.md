## Overview
Visc is a tool used for generating animated content, specifically videos, which can be used as website backgrounds or other dynamic elements. It allows users to upload references and generate images and animations.

## Role in this knowledge base
Used to create animated backgrounds for the website.

## Key facts
- Visc can generate video animations by uploading a reference and clicking 'animate'.
