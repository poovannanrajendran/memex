## Overview
Chasm VNC is an open-source tool that enables graphical access to applications running inside Docker containers via a web browser using VNC. It's developed by the company Chasm.

## Role in this knowledge base
Enables GUI access to containerized applications demonstrated in the video, such as web browsers and Obsidian.

## Key facts
- It facilitates browser-based access to containerized graphical applications.
