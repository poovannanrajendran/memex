## Overview
Make.com is a no-code Robotic Process Automation (RPA) platform that allows users to connect different software applications and automate workflows through a visual interface. It enables the creation of complex automation scenarios by visually chaining together various modules and services.

## Role in this knowledge base
This is the primary automation platform used to build the social media content generation system, orchestrating the flow between different AI tools and social media platforms.

## Key facts
- It is a no-code tool, meaning users can build automations without writing any programming code.
