## Overview
cal.com is an open-source scheduling tool that serves as an alternative to paid services like Calendly. It can be run using a Docker container, offering a free solution for users to share links for meeting arrangements.

## Role in this knowledge base
Presented as a cost-saving, open-source alternative for meeting scheduling, contrasting with paid solutions.

## Key facts
- Can be run with a Docker container for free meeting scheduling.
