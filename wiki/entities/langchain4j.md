## Overview
An open-source Java library specifically created to simplify the integration of Large Language Models (LLMs) for Java developers, providing a unified API to common LLMs and vector databases.

## Role in this knowledge base
Serves as an AI agent framework for Java environments, offering a modular design that supports agentic workflows and communication protocols.

## Key facts
- LangChain4j features a modular design and explicitly supports the Agent2Agent protocol.
