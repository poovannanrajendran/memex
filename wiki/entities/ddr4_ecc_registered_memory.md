## Overview
DDR4 ECC Registered (RDIMM) memory is error-correcting code memory commonly used in servers and workstations. ECC detects and corrects memory errors, improving system stability, while registered modules help support higher memory capacities.

## Role in this knowledge base
The type of RAM installed in the Dell Precision 7820, providing stability and capacity for the workstation.

## Key facts
- The build incorporated 64GB of DDR4 ECC Registered memory, costing around $70, although one stick later failed.
