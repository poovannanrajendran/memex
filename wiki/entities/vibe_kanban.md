## Overview
Vibe Kanban is a dashboard that enables users to run multiple AI coding agents concurrently. It functions like a project management tool for AI agents, allowing task assignment and progress monitoring from a single interface.

## Role in this knowledge base
Enhances ClaudeCode by enabling parallel processing of tasks through multiple AI agents, simulating a development team's workflow.

## Key facts
- Allows users to manage multiple AI coding agents at the same time from a single interface.
