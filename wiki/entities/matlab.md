## Overview
MATLAB is a proprietary multi-paradigm programming language and numerical computing environment developed by MathWorks. It is widely used for algorithm development, data visualisation, and scientific computing.

## Role in this knowledge base
Mentioned as one of the programming languages Bob uses for his quantitative analysis tasks.

## Key facts
- MATLAB does not run natively on Qualcomm Snapdragon processors with the ARM version of Windows and cannot be emulated effectively.
