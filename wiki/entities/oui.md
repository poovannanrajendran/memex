## Overview
OUI is a minimal UI library created by Kailash Nad, designed to be lightweight and free of common JavaScript bloat and dependencies found in many modern UI frameworks.

## Role in this knowledge base
A new UI library open-sourced by Zerodha's CTO, representing a departure from typical complex web development stacks.

## Key facts
- OUI requires only one CSS and one JavaScript file for integration and has zero dependencies.
