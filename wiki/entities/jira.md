## Overview
Jira is a popular project management and issue tracking tool used by software development teams to plan, track, and release software.

## Role in this knowledge base
The tool used for generating and managing user stories (tickets) which are then processed by the Coder AI.

## Key facts
- ChatGPT generated user stories formatted as Jira tickets for the development team.
