## Overview
A system maintenance and optimisation application for Mac computers, designed to clean up unnecessary files, manage performance, and enhance security.

## Role in this knowledge base
Highlighted as a key application for organising large amounts of files, keeping the Mac performing optimally, and acting as a 'mini personal assistant'.

## Key facts
- Its menu app provides real-time updates on Mac performance metrics like storage, memory usage, CPU load, temperature, Wi-Fi speeds, and battery health.
