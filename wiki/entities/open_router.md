## Overview
Open Router is a platform that acts as a gateway to over 300 AI models, allowing users to dynamically route requests to various LLMs. It provides visibility into model usage, costs, and token consumption.

## Role in this knowledge base
It is the core technology enabling the AI agent to dynamically select and switch between different language models.

## Key facts
- It allows users to access and route requests to over 300 different AI models.
