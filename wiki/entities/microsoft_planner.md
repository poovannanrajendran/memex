## Overview
Microsoft Planner is a task management and project planning tool designed for team collaboration within Microsoft 365.

## Role in this knowledge base
The recommended tool within Teams for managing complex projects, allowing for task delegation, deadlines, sub-tasks, and status tracking.

## Key facts
- Offers a Kanban-style board view ('group by progress') for visualising project status.
