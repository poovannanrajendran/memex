## Overview
Obsidian is a powerful knowledge base that works on top of a local folder of plain text Markdown files. It is the primary interface for Poovi's Second Brain.

## Role in this knowledge base
The container for the [[memex]] project. It allows for visualising connections between entities and concepts through its graph view.

## Key facts
- Uses local Markdown files.
- Supports heavy interlinking via [[wikilinks]].
- Primary tool for implementing the [[llm_wiki_pattern]].
