## Overview
The Cortex-X4 is a high-performance CPU core designed by Arm Holdings for premium smartphones and other devices requiring maximum processing power.

## Role in this knowledge base
Part of the heterogeneous CPU configuration in the NVIDIA Project Digits Mini PC, providing peak performance.

## Key facts
- Ten Cortex-X4 CPU cores are included in the Project Digits Mini PC's Arm-based CPU.
