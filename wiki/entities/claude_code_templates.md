## Overview
Claude Code Templates are pre-built configurations for ClaudeCode, offering over 100 ready-made agents, commands, automations, and workflows. These templates can be quickly imported to set up agents, significantly reducing initial configuration time.

## Role in this knowledge base
Provides ready-to-use components for ClaudeCode, acting like an app store that accelerates the setup and deployment of AI agents and workflows.

## Key facts
- Offers over 100 ready-made setups that can be plugged straight into Claude Code.
