## Overview
An open-source text-to-image diffusion model known for generating detailed images based on text prompts. It is widely used in AI art generation and research.

## Role in this knowledge base
Used as a specific tool to test the quality and accuracy of AI responses between the base M4 Mac Mini and the M4 Pro model, particularly in image generation.

## Key facts
- The M4 Pro model provided more accurate image generation results with Stable Diffusion compared to the base model using the exact same prompts.
