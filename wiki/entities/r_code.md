## Overview
R Code (likely a misspelling or alternative name for 'Rode Code' or similar VS Code extension) is a VS Code extension that allows for overriding system prompts. It includes modes like 'architect', 'ask', and 'debug' and is used for planning and implementing code changes.

## Role in this knowledge base
The presenter uses R Code's 'architect' or 'ask' mode for planning large code changes and importantly, uses it to implement overridden prompts ('coder short rules') to save on API costs.

## Key facts
- R Code is a VS Code extension that allows users to override system prompts with custom ones like 'coder short rules'.
