## Overview
An open-source orchestration framework that enables developers to define AI agents as a 'crew' of 'workers,' assigning them specialized roles, goals, and backstories using natural language.

## Role in this knowledge base
Highlighted as a popular AI agent framework for orchestrating teams of agents to complete complex tasks, supporting various LLMs and RAG tools.

## Key facts
- CrewAI supports both sequential and hierarchical task processes for its agent crews.
