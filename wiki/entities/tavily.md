## Overview
Tavily is a web search engine API designed specifically for agents, providing search results in a structured format.

## Role in this knowledge base
It is used as a tool within the agentic RAG system for performing web searches when necessary, such as during self-correction or when retrieved local documents are insufficient.

## Key facts
- It functions similarly to services like You.com or Perplexity AI, offering natural language search capabilities for agents.
